# Contributor Covenant Code of Conduct

## Our pledge

We pledge to make participation in SSC a harassment-free experience for everyone, regardless of background or experience level.

## Expected behavior

- Be respectful and constructive in issues, PRs, and discussions
- Accept feedback on code and design
- Focus on what is best for the project and users' privacy

## Unacceptable behavior

- Harassment, insults, or personal attacks
- Publishing others' private information without permission
- Posting production secrets or exploit details in public threads (use [SECURITY.md](SECURITY.md) instead)

## Enforcement

Report concerns to the repository owner via GitHub or **contact@supersecurechat.com**. Maintainers may remove or close contributions that violate this code.

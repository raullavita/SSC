TASK P — Founder infra checklist (supersecurechat.com)
======================================================
Run verification anytime:  .\scripts\verify_task_p.ps1

P.1 — Cloudflare Turnstile (free)
  1. https://dash.cloudflare.com → Turnstile → Add site
  2. Hostnames: supersecurechat.com, www.supersecurechat.com, localhost
  3. Copy Site Key + Secret Key

P.2 — Paste keys (then redeploy API + rebuild clients)
  backend/cloud_run.env:
    TURNSTILE_SITEKEY=...
    TURNSTILE_SECRET=...
  frontend/.env.production.local:
    REACT_APP_TURNSTILE_SITEKEY=...   (same site key)
  Then:
    .\scripts\deploy_cloud_run.ps1
    cd frontend && yarn build && cd .. && firebase deploy --only hosting

P.3 — API custom domain api.supersecurechat.com
  Google Cloud Console → Cloud Run → ssc-api → Manage custom domains
  OR: gcloud run domain-mappings create --service=ssc-api --domain=api.supersecurechat.com --region=europe-west1 --project=super-chat-b0992
  Cloudflare DNS (supersecurechat.com zone):
    Add CNAME api -> ghs.googlehosted.com (or records Google shows)
  Wait for SSL provisioning (up to 24h, often minutes)

P.4 — Google OAuth consent + credentials
  https://console.cloud.google.com/apis/credentials (project super-chat-b0992)
  Authorized domains: supersecurechat.com
  OAuth client → Authorized redirect URIs (keep existing + add when API domain live):
    https://ssc-api-4jp3wuccwa-ew.a.run.app/api/auth/google/callback
    https://api.supersecurechat.com/api/auth/google/callback
  Publish consent screen if still in Testing

P.5 — After api.supersecurechat.com is live (optional migration)
  backend/cloud_run.env:
    GOOGLE_REDIRECT_URI=https://api.supersecurechat.com/api/auth/google/callback
  Redeploy API. APK/desktop can keep Cloud Run URL until next rebuild.

P.6 — TURN off-LAN proof (manual)
  dots on cellular data, smashmaxxx on Wi-Fi (or reverse)
  Voice + video call both directions. Log result in test_reports/turn_off_lan_YYYY-MM-DD.md

P.7 — Email contact@supersecurechat.com (Porkbun hosted)
  Porkbun → Domain → Email → contact@ inbox (paid)
  Click "Fix Email Hosting DNS Records" for SPF/DKIM/DMARC
  See scripts/PORKBUN_EMAIL_SETUP.txt

P.8 — Desktop code signing (Q.61 — optional, paid)
  Windows: OV/EV Authenticode cert → CSC_LINK + CSC_KEY_PASSWORD → signed NSIS
  Mac: Apple Developer → notarize (APPLE_ID, APPLE_APP_SPECIFIC_PASSWORD, APPLE_TEAM_ID)
  Full guide: scripts/CODE_SIGNING_SETUP.txt
  Signed build: .\scripts\build_desktop_signed_win.ps1  or  ./scripts/build_desktop_signed_mac.sh
  After first signed release: .\scripts\enable_code_signing.ps1 + redeploy API flags

P.9 — Play Console public listing (Q.62 — $25 one-time)
  Full guide: scripts/GOOGLE_PLAY_SETUP.txt
  Listing copy: play-store/LISTING_COPY.md
  Validate assets: .\scripts\prepare_play_listing.ps1
  After live: .\scripts\enable_play_store.ps1 -StoreUrl "https://play.google.com/store/apps/details?id=chat.ssc.secure"

P.10 — iOS App Store (Q.63 — $99/yr Apple Developer, requires Mac)
  Full guide: scripts/APP_STORE_SETUP.txt
  Listing copy: app-store/LISTING_COPY.md
  Validate: .\scripts\prepare_app_store_listing.ps1  or  ./scripts/prepare_app_store_listing.sh
  Build on Mac: ./SSC-BUILD-IOS.sh → Xcode Archive → TestFlight → App Store
  After live: .\scripts\enable_app_store.ps1 -StoreUrl "https://apps.apple.com/app/idXXXXXXXXX"

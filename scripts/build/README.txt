SSC build entrypoints (canonical scripts live here).

Windows (double-click or cmd):
  SSC-BUILD-APK.bat           -> scripts/build/build_apk.bat
  SSC-BUILD-DESKTOP-WIN.bat   -> scripts/build/build_desktop_win.bat

macOS / Linux:
  ./SSC-BUILD-DESKTOP-MAC.sh  -> scripts/build/build_desktop_mac.sh
  ./SSC-BUILD-IOS.sh          -> scripts/build/build_ios.sh

Signed desktop variants: scripts/build_desktop_signed_win.ps1, build_desktop_signed_mac.sh

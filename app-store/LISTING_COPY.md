## Description

SSC (Super Secure Chat) is a privacy-first messaging app for people who want encrypted conversations that disappear by default.

**Highlights**
- End-to-end encryption with libsignal (post-quantum hybrid on supported platforms)
- Ephemeral by default — messages and files auto-delete (24 hours)
- Voice and video calls
- Groups, reactions, and security controls (app lock, 2FA, panic wipe)
- No browser chat — install the app to register and sign in

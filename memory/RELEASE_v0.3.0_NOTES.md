# SSC v0.3.0

First release with **attached Electron installer** and **Android APK** (Steps 10–17).

## Downloads

- **Windows:** `SSC-Setup-0.3.0.exe` — Electron + libsignal
- **Android:** `SSC-0.3.0.apk` — WebView shell + libsignal-android 0.96.4

## Highlights

- Usernames + invite links (`@username`, `/add/{username}`)
- Read receipts UI with per-chat privacy overrides
- Call reliability (ICE queue, hangup/missed flow)
- Message edit, delete, forward
- Safety numbers + trust UX
- Multi-device QR linking (`ssc://link-device`)
- Encrypted client backup (`.ssc-backup`)
- Android shell polish (splash, deep links, pull-to-refresh, offline retry)
